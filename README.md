# 💰 Smart Finance App

An AI-Powered Personal Finance Management System built using React, Firebase, OCR, and AI technologies.

---

## 🚀 Features

### 🔐 Authentication
- User Signup
- User Login
- Email Verification
- Secure Firebase Authentication

### 💰 Wallet Manager
- Add Money to Wallet
- Track Wallet Balance
- Wallet Transaction History
- Income Source Tracking

### 💸 Expense Management
- Add Expenses
- Edit Expenses
- Delete Expenses
- Category-Based Tracking
- Expense History

### 📅 Expense Calendar
- Calendar View
- Date-wise Expense Tracking
- Daily Expense Summary
- Expense Heatmap Visualization

### 📷 Smart Receipt Scanner
- OCR Receipt Scanning using Tesseract.js
- Auto Detect Amount
- Auto Detect Purpose
- Auto Detect Category
- Manual Editing Support

### 📊 Expense Analytics
- Expense Charts
- Category Analysis
- Spending Trends
- Financial Insights

### 🤖 AI Financial Assistant
- AI-Powered Financial Guidance
- Spending Suggestions
- Expense Analysis
- Personalized Advice

### 🎯 Savings Goals
- Create Savings Goals
- Track Goal Progress
- Goal Completion Monitoring

### 💵 Lending Manager
- Money Given Tracking
- Money Borrowed Tracking
- Interest Calculation
- Repayment Monitoring

### 🔔 Reminder Center
- Create Custom Reminders
- Notes Support
- Future Date Validation
- Edit Reminder
- Delete Reminder
- Notification Alerts

### 📄 PDF Reports
- Export Complete Finance Report
- Expense Summary
- Wallet Summary
- Income Summary

## ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/saimanraj/SmartFinanceApp.git
```

Move to Project Folder

```bash
cd SmartFinanceApp
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Build Project

```bash
npm run build
```
